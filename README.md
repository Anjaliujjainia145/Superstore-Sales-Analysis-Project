# Superstore Sales Analysis

## Project Overview
This project analyzes a retail sales dataset to identify key business insights related to sales, profit, customer segments, regions, and product categories.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Technology category generates the highest sales and profit
- West and East regions dominate total revenue
- Consumer segment contributes more than 50% of total sales
- Standard Class shipping generates the highest sales

## Business Recommendations
- Focus on high-profit Technology products
- Improve Furniture profitability
- Reduce excessive discounts in low-profit regions# Superstore-Sales-Analysis-Project
